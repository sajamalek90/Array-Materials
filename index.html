Array Fundamentals

Array Introduction

Arrays are ordered collections of values that can hold multiple items of any type in a single variable.

// Creating arrays
let fruits = ['Apple', 'Banana', 'Orange'];
let mixedData = [1, 'Hello', true, {name: 'John'}];

Arrays are zero-indexed: the first element is at position 0.

Using Length With Array

The length property returns the number of elements in an array.

let colors = ['Red', 'Green', 'Blue'];
console.log(colors.length); // 3

// Changing length
colors.length = 2;
console.log(colors); // ['Red', 'Green']

Array Methods

Add And Remove From Array

Common methods for modifying arrays:

let numbers = [1, 2, 3];

// Add to end
numbers.push(4); // [1, 2, 3, 4]

// Remove from end
numbers.pop(); // [1, 2, 3]

// Add to start
numbers.unshift(0); // [0, 1, 2, 3]

// Remove from start
numbers.shift(); // [1, 2, 3]

Searching Array

Methods to find elements in arrays:

let languages = ['JavaScript', 'Python', 'Java', 'C++'];

// indexOf - finds first occurrence
console.log(languages.indexOf('Java')); // 2

// includes - checks existence
console.log(languages.includes('Ruby')); // false

// find - finds first match
let result = languages.find(lang => lang.length > 5);
console.log(result); // 'JavaScript'

Sorting Array

The sort() method sorts elements as strings by default.

let fruits = ['Banana', 'Orange', 'Apple'];
fruits.sort(); // ['Apple', 'Banana', 'Orange']

let numbers = [40, 100, 1, 5, 25];
numbers.sort((a, b) => a - b); // [1, 5, 25, 40, 100]

Slicing Array

slice() extracts a portion of an array without modifying the original.

let colors = ['Red', 'Green', 'Blue', 'Yellow', 'Purple'];

// Extract elements from index 1 to 3 (not including 3)
let subset = colors.slice(1, 3); // ['Green', 'Blue']

// Copy entire array
let copy = colors.slice();

Joining Arrays

Combine arrays or convert them to strings:

let arr1 = [1, 2, 3];
let arr2 = [4, 5, 6];

// Concatenate arrays
let combined = arr1.concat(arr2); // [1, 2, 3, 4, 5, 6]

// Join elements to string
let str = arr1.join('-'); // "1-2-3"

Array Challenges

Array Challenge

Create a function that takes an array of numbers and returns:

  1. The sum of all numbers
  2. The average of the numbers
  3. A new array with only even numbers
function processNumbers(numbers) {
    // Sum
    const sum = numbers.reduce((total, num) => total + num, 0);
    
    // Average
    const average = sum / numbers.length;
    
    // Even numbers
    const evens = numbers.filter(num => num % 2 === 0);
    
    return { sum, average, evens };
}

const result = processNumbers([1, 2, 3, 4, 5]);
console.log(result);

Output: { sum: 15, average: 3, evens: [2, 4] }
